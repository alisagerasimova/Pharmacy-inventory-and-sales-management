# Pharmacy-inventory-and-sales-management
Курсовая. ПМ 01. Разработка модулей программного обеспечения для компьютерных систем.
**Тема:** Разработка модуля «Учет медикаментов и продаж в аптеке»

## Стек
* **Язык:** Python
* **Библиотека GUI:** Tkinter
* **БД:** SQLite

## Функционал
* Учет поступления медикаментов на склад.
* Автоматический мониторинг сроков годности.
* Оформление и регистрация актов продажи.
* Поиск товаров по справочнику (наименование).
